## Development
### Linting 
Linting is done via autopep8
```bash
script/lint.sh
```

### Documentation
Generates to [./docs](docs) (as one part of build process).  See [build script](script/build.sh) for more info.
```bash
script/build.sh
```

### Testing
```bash
  clear; pytest
```

### Building
`script/build.sh`
